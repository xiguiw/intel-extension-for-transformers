class intel_extension_for_transformers.neural_chat.tools.rome.tok_dataset.TokenizedDataset(text_dataset, tokenizer=None, maxlen=None, field='text')[source]

Converts a dataset of text samples into a dataset of token sequences, as converted by a supplied tokenizer. The tokens come along with position ids and attention masks, they can be supplied directly to the model.

intel_extension_for_transformers.neural_chat.tools.rome.tok_dataset.length_collation(token_size)[source]

Sorts a batch of sequences and breaks it up into subbatches of same-sized sequences, padding as needed. Each batch has no more than token_size total tokens (or a single sequence, if the sequence happens to be larger).
